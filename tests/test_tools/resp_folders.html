Une question ? Besoin d'aide ?

Comment supprimer, déplacer ou partager mon fichier ?
Pour afficher les options d'un fichier, faites un clic-droit sur le fichier souhaité. Pour obtenir les "codes de partages", cliquez sur Partager dans ce même menu.

Comment supprimer, déplacer ou télécharger (récupérer sur votre ordinateur) plusieurs fichiers à la fois ?
Cliquez sur l'icône située en haut à droite au dessus de vos fichiers. Vous basculerez alors vers le mode listage, permettant d'effectuer des opérations multiples.

Comment modifier l'ordre d'affichage de mes dossiers ou de mes fichiers ?
Rendez-vous dans la section Paramètres puis dans la rubrique Manager Fichiers, puis sélectionner l'ordre de listage des dossiers ou des fichiers

Je souhaite voir le poids ou bien la date d'envoi de mon fichier
Rendez-vous dans la section Paramètres puis dans la rubrique Manager Fichiers, cochez les cases correspondants aux informations que vous souhaitez afficher.

Je rencontre un problème avec le formulaire d'envoi de fichiers
Rendez-vous dans la section Paramètres puis sélectionnez un autre Uploader dans la liste.

Je n'ai pas obtenu de réponse à ma question. Je constate un bug.
Dans ce cas, nous vous invitons à contacter le Support
